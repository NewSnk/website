designing organic shapes for additive manufacturing

after learning rhino and grasshopper i wanted to leverage tools to create a generative design process. the process allows me to insert a mesh, as well as any solid components, and any empty space that must be maintained. the grasshopper process i designed allows you to either manually customize a rendering or go through an evolutionary solver, selecting preferred designs that become parent genomes for the new generations.

i designed a grasshopper file that allows for the form to change in node size, beam size, point count, seed, point distribution, and neighest neighbor calculations.

i perform beam and node structural analysis and grade the designs on their fill (how well the form matches the contours of the original mesh), the volume, and the strength.

the form is then passed through an evolutionary solver that allows the user to select favorite designs and those genes are passed to ensuring generations.
